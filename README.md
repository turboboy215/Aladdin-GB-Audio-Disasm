Aladdin (Game Boy) audio disassembly

A disassembly of the sound engine and data from Aladdin for Game Boy (the monochrome version from 1994), fully labeled and commented. Can be built with RGBDS; the sound data of the resulting ROM is verified to match the sound data from the original ROM exactly. However, the resulting ROM itself on its own is not playable, as no additional code is included.
A text file is also included which documents how the music, sound effect, and instrument data is structured for this game as well as other games using the same sound engine.
